# Prien Aktiv PreHab

Eine einfache, datenschutzfreundliche Streamlit-App zur Vorbereitung auf Knie- und Hüftendoprothetik.

## Lokal starten

```bash
pip install -r requirements.txt
streamlit run app.py
```

## GitHub und Veröffentlichung

1. Erstelle ein neues Repository auf GitHub.
2. Lade diesen Ordner hoch.
3. Öffne Streamlit Community Cloud und verbinde dein Repository.
4. Wähle die Datei app.py als Einstiegspunkt.
5. Setze das Passwort über die Streamlit-Cloud-Secrets oder über eine Umgebungsvariable.

## Hinweis

Die App ist bewusst ohne Patientendaten-Speicherung gebaut. Für die echte Nutzung sollten Datenschutz und Rechtslage separat geprüft werden.
